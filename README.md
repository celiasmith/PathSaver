PathSaver
=========
Shows finger locations and traces their trajectories on touchscreen. Notes
 1) Allows up to 4 fingers to be tracked.
 2) Allows you to save the (x,y) points to a .txt file in Downloads
 3) Based on "TouchPaint.java" provided by Android samples
 4) based on touchscreen_Tester by R Kakarala: Started 21 July 2011
